# Spiced Gracken [![Build Status](https://travis-ci.org/NullVoxPopuli/spiced_gracken.svg)](https://travis-ci.org/NullVoxPopuli/spiced_gracken)
a [RUM](https://github.com/NullVoxPopuli/Rum) client in Ruby

# Installation

```bash
gem install spiced_gracken
```

# Usage

```bash
$ irb
```
```ruby
require 'spiced_gracken'
SpicedGracken.start
```
